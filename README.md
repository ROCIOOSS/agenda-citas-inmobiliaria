# 🏡 Agenda de Citas - Consultoría Inmobiliaria

Aplicación web responsive para agendar citas con una empresa de consultoría inmobiliaria que asesora a inversionistas mexicanos interesados en bienes raíces residenciales en Arizona y Texas.

## 📋 Funcionalidades

- Diseño moderno, profesional y amigable
- Secciones informativas: ¿Quiénes somos?, Misión, Visión, Valores y Servicios
- Acordeones interactivos con información sobre propiedades y ubicaciones
- Formulario de agendamiento con:
  - Validación de campos requeridos
  - Envío directo a Google Sheets mediante Apps Script
  - Confirmación visual de éxito
  - Scroll automático y limpieza del formulario

## 🧠 Tecnologías utilizadas

- HTML5
- CSS3 (embebido)
- JavaScript (embebido)
- Google Apps Script para conexión con Google Sheets

## ✅ Cómo usar

1. Haz clic en “Use this template” o clona este repositorio.
2. Carga el archivo `.html` en tu servidor o activa GitHub Pages.
3. Asegúrate de tener un Apps Script vinculado con tu Google Sheet para recibir los datos del formulario.
4. Personaliza el logo, imágenes y textos si lo deseas.
